ibus没办法切换到rime输入法的问题

Gentoo中新安装ibus-rime-1.2后，结果没办法切换到新安装的rime输入法。

经过一番搜索，发现workaround如下。

  • 首先尝试手动切换输入法：1 

    /usr/libexec/ibus-rime/ibus-engine-rime

    命令行报错： 

    error while loading shared libraries: libopencc.so.1: cannot open shared object file: No such file or directory

  • /usr/lib64 下面确实没有libopencc.so.1文件2。重新建立一个链接文件： 

    ln /usr/lib64/libopencc.so.1.0.0 /usr/lib64/libopencc.so.1

问题解决。

Footnotes:

1

有可能ibus-engine-rime文件位置会不一样

2

这可能是因为我把opencc更新到了最新版1.0.2
