windows上设置预览org-mode中的化学结构式

之前一直没办法在windows平台上面预览org-mode中用latex画的结构式。最近总算有时间找到了原因。

正确的步骤如下：

  • 安装latex，并确认在$PATH中。我选择mitex1。
  • 安装imagemagick。

  • 安装ghostscript2，并确认在$PATH中。非常重要。之前我就是遗漏了这一步，导致一直报错： 

    org-create-formula-image-with-imagemagick: Failed to create png file
from c:/DOCUME~1/ADMINI~1/LOCALS~1/Temp/orgtex2966TN.tex
  • 其他设置参见 在org-mode中显示化学结构式

Footnotes:

1

http://www.mitex.org

2

ImageMagick For Windows ghostscript用于pdf转png，正是org-create-formula-image-with-imagemagick需要的。 ImageMagick Errors: Convert PDF to Images - Stack Overflow
